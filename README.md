# runscope-export-import

Java command line tool to export / import data from / to https://www.runscope.com/ in json format.
 

## Requirements
 - Java 12
 - Runscope account & API key

## Available commands
 - help
 - setapikey key
 - account
 - export
 - export bucketKey1 bucketKey2...
 - export 'param'
 - import
 - import bucketKey1 bucketKey2...
 - import 'param'
 - settings
 - quit
